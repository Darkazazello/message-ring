Hello world example

To try this example, you need GNU make and git in your PATH.

To build the example, run the following command:

$ make

To start the release in the foreground:

$ ./_rel/bin/hello_world_example console

Then point your browser at http://localhost:8080. Create ring operation http://localhost:8080/create?n=300 View active rings http://localhost:8080/list Push message to ring http://localhost:8080/run?id=&m=100 View log http://localhost:8080/log
